About

Current Version: 1.0

The NerdFarm is a low cost, hydroponic growth chamber; designed as a simplified version of the MIT OpenAg Personal Food Computer. While this is a working hydroponic environment, the focus in on collecting research data. This is an open source project based around a Raspberry Pi 'brain' and Python code, with a CouchDB repository.

Documentation is on the FutureAg Wiki

Discussion is on the MIT OpenAg Forum

Code is sourced on Github
